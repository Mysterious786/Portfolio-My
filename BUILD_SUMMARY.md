## 🎬 Animation System

### GSAP Features Used
- ✅ **fromTo()** - Entrance animations
- ✅ **ScrollTrigger** - Scroll-triggered animations
- ✅ **Stagger** - Sequential item animations
- ✅ **Event listeners** - Hover effects
- ✅ **Continuous animations** - Floating elements

### Animation Types
```
Entrance Animations:
├── Title slide up + fade (0.8s)
├── Buttons slide up + fade (0.8s)
├── Cards fade in + scale (0.8s)
└── Floating boxes float animation (3-5s)

Hover Animations:
├── Cards lift (y: -10px, 0.4s)
├── Images zoom (scale: 1.1, 0.5s)
├── Icons rotate (10deg, 0.4s)
├── Links color change (0.3s)
└── Overlays fade in (0.4s)

Scroll Animations:
├── Sections trigger at 80% scroll
├── Staggered card entrance (0.15s delay)
├── Underlines animate width
└── Toggle on scroll enter/leave
```
